A How To Guide for the data.nasa.gov API

by Philip Lewallen


Overview of this guide

This guide is for the data.NASA.gov API. The will cover the available functions for the API for www.data.nasa.gov, and how the data returned is formatted. Lastly it will provide examples on how to iterate through the JSON data and display it. On the left hand columns of this descrtiption there are links to the documentation on the methods of the API. At the bottom of each page there is going to be a link to the next page in the guide.


What is "data.nasa.gov"?

data.nasa.gov is a project by NASA to make all of the NASA's data available to the public for research purposes. According to the home page of data.nasa there is a alot of data available. One example of how much data NASA has is just one mission from the Earth Observing System Data and Information Systems (EOSDIS) has stored mroe than 3 petabytes of data since 2005. This api was developed to track the changes in this data.


What does the data.nasa.gov API contain

It contains 9 methods. These methods are explained later on in this guide.If you would like to see NASA's own documentation on the subject it can be found here. What do these methods do exactly?

These methods do many things. Well one thing actually. They return a JSON object listing the information regarding the changed data from your search parameters. That is it. Only the changes.


Why are the changes in the data important?

Because there is so much data available you couldn't download or search the database effecitvely. So this list of data and it's changes provides the lightest footprint possible for the user to navigate and disseminate for use. And there is a lot of data in these change lists. We will go over all of the different data fields in the JSON objects the methods return later in this guide.
